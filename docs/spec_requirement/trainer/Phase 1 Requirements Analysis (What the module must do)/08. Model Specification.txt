Purpose: Defines trainer data models and responsibilities.

Model Entities:

1. Trainer Model
   - Table: trainer
   - Attributes: trainer_id, full_name, email, phone, skills, qualifications, experience, status, created_at, updated_at, created_by, updated_by.
   - Operations: Insert, update, query all, query by id, delete.

2. Trainer Availability Model
   - Table: trainer_availablity
   - Attributes: availability_id, trainer_id, date, status, availability_type, duration_type, time_slot, start_time, end_time, description, created_at, updated_at, created_by, updated_by.
   - Operations: Fetch availability records for trainer, insert override, update override, delete override.

3. Calendar Event Model
   - Table: calendar_event
   - Attributes: event_id, date, event_type, description, created_at, updated_at, created_by, updated_by.
   - Operations: Fetch all events, insert event, delete event.
